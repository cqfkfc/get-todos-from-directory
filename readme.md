Live Example at https://todos-cqf.vercel.app/

# Set up

## To start the server (NodeJS)

`cd server` `npm install` `npm start`

## To start the client (ReactJS)

`cd client` `npm install` `npm start`

## To test the client (Jest)

`cd client` `npm test`
